how to use, type in terminal:

> python championship.py

default championship is Italian Serie A, the teams are in serie.txt, with value 
for each team, in this format:

Team Name:Team value

in championship.py you can find many parameters at the beginning of the file.

calendar.py is for fixtures generation, given a list of teams in input.
matchplay.py is for simulating the match between two teams, given their name and value.
